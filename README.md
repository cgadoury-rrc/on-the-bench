# OnTheBench
>An Android app to follow you favourite NHL teams and players, 
>complete with up-to-date rosters, games, and standings. Created with 
>Kotlin and Jetpack Compose.

## Features
- Daily NHL game scores
- Current NHL standings
- Up-to-date NHL rosters
- **Player Cards** (statistic visualizations) with charts and graphs
